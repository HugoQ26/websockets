# Chat app

#### Tech stack: Node.js, Express.js, Socket.io, Js

#### The idea was:

1. to make simple chat app
2. use socket.io
3. no frontend libraries, vanilla javascript

Not yet deployed.

#### Installation

First clone repo.

Next in root folder:

`npm install` - this will install all dependencies

And then run:

`npm start` - this will start Node server with app.

And in the browser:

`http://localhost:8000/`
